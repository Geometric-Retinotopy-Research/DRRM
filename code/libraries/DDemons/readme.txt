Demons Registration
Herve Lombaert, 2011

------------------------------------------------------------------
Description:

Matlab code for 2D and 3D Image Registration 
using the Diffeomorphic Log-Demons Algorithm

Code is provided in order to help the understanding
of the Demons algorithm

------------------------------------------------------------------
Instructions:

demons2d/runme.m  - entry point, start and run file
demons3d/demons.m - entry point, start and run file

------------------------------------------------------------------
Bibliography:

[1] Diffeomorphic demons: Efficient non-parametric image registration,
    Vercauteren, Pennec, Perchant, Ayache -- NeuroImage 2009, (45)1:61-72
[2] Symmetric Log-Domain Diffeomorphic Registration: A Demons-Based Approach
    Vercauteren, Pennec, Perchant, Ayache -- MICCAI 2008, 754-761

For more recent work/survey, exploiting global shape characteristics
(instead of the conventional local gradient-based approaches), consider reading

[1] Spectral Log-Demons: Diffeomorphic Image Registration with Very Large Deformations
    Lombaert, Grady, Pennec, Ayache, Cheriet -- IJCV 2014, (107)3:254-271
